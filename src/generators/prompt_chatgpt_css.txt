По заданным компонентам JSON нужно сгенерировать CSS файл. Название классов написано в поле classname. Пример:

/* Alert */
.alert-success {
    background-color: #dff0d8;
    border-color: #d6e9c6;
    color: #3c763d;
}

/* Button */
.btn-primary {
    background-color: #337ab7;
    border-color: #2e6da4;
    color: white;
}
